# Koala Blog

Koala is a SaaS Blog App and still work in progress.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```bash
cd code
```
```bash
docker-compose up
```


```bash
docker exec -it cphp bash
```

```bash
composer install
```

```bash
chmod -R 777 var/
```
